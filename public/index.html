TikTok Live Overlay

เครื่องมือสำหรับสตรีมเมอร์ TikTok Live — แสดงของขวัญ แชท ยอดไลค์ และอ่านแชทเสียงบน TikTok Live Studio, OBS แบบเรียลไทม์ ไม่ต้องติดตั้งโปรแกรมเพิ่ม แค่คัดลอกลิงก์ไปใส่ Browser Source

เริ่มต้นใช้งาน — เข้าสู่ระบบด้วย Google

ข้อมูลบัญชี TikTok ของคุณใช้เพื่อเชื่อมต่อไลฟ์กับระบบ Overlay เท่านั้นไม่ได้จัดเก็บข้อมูลใดๆ

ทดสอบบน localhost: ต้องเพิ่ม http://localhost:3000 ใน Google Cloud Console → Credentials → Authorized JavaScript origins

ใช้งานง่าย 3 ขั้นตอน

ตั้งค่าครั้งเดียว ใช้ได้ทุกไลฟ์ — รองรับ OBS, TikTok Live Studio และโปรแกรมสตรีมอื่นๆ

1

Login ด้วย Google

เข้าสู่ระบบเพื่อเปิด Dashboard ส่วนตัวของคุณ ปลอดภัยและรวดเร็ว

2

กรอก TikTok Username

ใส่ชื่อผู้ใช้ TikTok แล้วคัดลอกลิงก์ Overlay ที่ต้องการ — ไม่ต้องรอเชื่อมต่อก็ตั้งค่า TikTok Live Studio, OBS ล่วงหน้าได้

3

วางใน TikTok Live Studio, OBS แล้วไลฟ์

เพิ่ม Browser Source ตั้งพื้นหลัง Transparent ของขวัญและแชทจะอัปเดตอัตโนมัติเมื่อมีคนส่งของ

ตัวอย่าง Overlay

ดูตัวอย่าง Widget ยอดนิยมก่อน Login — ปรับสี โถ และรูปแบบได้เต็มที่หลังเข้า Dashboard

ฟีเจอร์หลัก

เลือก Overlay ได้ตามสไตล์ไลฟ์ของคุณ — ปรับแต่งโถ สี พื้น และเสียง TTS ได้จาก Dashboard

  • 🎁 Gift Jar ฟิสิกส์ ของขวัญตกลงโถด้วยฟิสิกส์จริง เลือกโถได้หลายแบบ (แก้ว Mason, มังกรทอง, Ultra ฯลฯ)
  • 🌿 Gift Floor ของขวัญกองบนพื้น — เหมาะกับไลฟ์สไตล์เกมหรือฉากพื้นหญ้า/พื้นเค้ก
  • 🍱 Gift ปิ่นโต ของขวัญเรียงในปิ่นโตหลายชั้น สไตล์ไทยๆ โดดเด่นบนจอไลฟ์
  • 💬 Chat Overlay แสดงข้อความแชทผู้ชมแบบเรียลไทม์ ปรับพื้นหลังโปร่งใสหรือดำอ่อนได้
  • 🔊 Chat TTS อ่านแชทผู้ชมเป็นเสียง Neural ออนไลน์ เลือกเสียง ความเร็ว และรูปแบบการอ่านได้
  • ❤️ Like Hearts & Ranking หัวใจลอยเมื่อมีคนกดไลค์ พร้อม Top Likers และ Top Gifters อัปเดตสด
  • 🎯 Live Goal ตั้งเป้าหมายเหรียญไลฟ์ แสดงยอดสะสมและแถบเปอร์เซ็นต์ให้ผู้ชมติดตามความคืบหน้า
  • อื่นๆ...
