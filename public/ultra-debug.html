Ultra Jar — เส้นขอบชนปัจจุบัน

รูปนี้วาดทับบน ultra.png ขนาด stage จริง 460×620 (เท่า overlay) เทียบกับเส้น marking เดิม (น้ำเงิน) กับ config ปัจจุบัน (สีอื่น)

คำอธิบายสี

  • น้ำเงิน = marking เดิม (อ้างอิง)
  • แดง = ขอบแก้ว config ปัจจุบัน
  • ส้ม = กำแพงชนจริง (+4px inset)
  • เขียว = ช้อน (ramp + guard)
  • ฟ้า = feedZone (ไม่นับเป็นล้น)
  • เหลือง = จุดเกิด spawn
  • ม่วง = ปลายช้อน (tip)

สร้างจาก node scripts/ultra-debug-overlay.js
ดู live: ?debug=1

สิ่งที่เปลี่ยนจาก marking เดิม (น้ำเงิน):
• ช้อน (เขียว) ปรับล่าสุด — ปลายช้อน (ม่วง) y≈360, ramp center y≈273
• จุด spawn (เหลือง) ขยับซ้าย — centerX 374 → 328
• ไม่มี chute / rim wall แล้ว — ปากโถเปิดด้านบน
• กำแพงส้มเริ่มที่ bodyTopY=362 (ไม่มีขอบด้านบนปิด)
